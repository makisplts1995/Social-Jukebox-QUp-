Social Jukebox Mock (QUp)
AM me25045-me25054-me25081-me25095

ΠΩΣ ΑΝΟΙΓΕΙ:
Ανοίξτε το αρχείο index.html σε κάποιον browser.

ΤΙ ΝΑ ΚΑΝΕΤΕ:

1. Guest (Θαμώνας)
   - Πατήστε "Guest / Visitor" -> "Simulate Scan".
   - Ζητήστε τραγούδια ("QUp it") και ψηφίστε ("Up Next").
   - Δείτε τον Player κάτω.

2. Host (Ιδιοκτήτης)
   - Πατήστε "Host / Owner" -> "Sign In".
   - Πατήστε "New Session" -> "Start Party".
   - Δείτε τα αιτήματα (Requests) και ελέγξτε τη μουσική.

3. Admin (Διαχειριστής)
   - Πατήστε το κουμπί "Admin" (λουκέτο) στην αρχική.
   - Δείτε τα στατιστικά του συστήματος.
